🚀 COMO COMEÇAR

1. VERIFICAR SAÚDE DO SISTEMA
   → Acesse: https://seu-dominio.com/health_check.php
   → Você verá dashboard com pontuação do sistema

2. ACESSAR O DASHBOARD
   → Acesse: https://seu-dominio.com/public/dashboard_professional.php
   → Interface moderna e responsiva

3. LER A DOCUMENTAÇÃO
   → Comece com: /docs/REFERENCIA_RAPIDA.md
   → Depois: /docs/GUIA_PROFISSIONAL.md
   → Para desenvolvimento: /docs/GUIA_DESENVOLVIMENTO.md

4. CRIAR SUA PRIMEIRA PÁGINA
   Copie este código:
   
   <?php
   require_once '../core/init_professional.php';
   requireAuth();
   
   $layout = new ProfessionalLayout('Minha Página');
   $layout->addBreadcrumb('Home', '/');
   $layout->setContent('<div class="card"><div class="card-body">Conteúdo</div></div>');
   
   echo $layout->render();
   ?>

5. MIGRAR PARA PRODUÇÃO
   → Siga: /docs/DEPLOYMENT_CHECKLIST.md
   → Verifique todos os itens antes de deploy

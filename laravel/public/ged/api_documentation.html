📡 Documentação da API REST - E-Doc

⚠️ Importante: Esta API é para uso interno e requer autenticação via API Key no header X-API-Key.

🔑 Autenticação

Envie o header X-API-Key com uma das seguintes chaves:

  • edoc-api-key-2024 (produção)
  • test-key-123 (desenvolvimento)

📋 Endpoints Disponíveis

📄 Documentos

GET
/api.php/documents

Listar todos os documentos com filtros opcionais

Parâmetros de Query:

  • status - Filtrar por status
  • search - Busca por título ou conteúdo
  • limit - Limite de resultados (padrão: 50)

Exemplo:

GET /api.php/documents?status=Aprovado&limit=10
Headers: X-API-Key: edoc-api-key-2024
GET
/api.php/documents/{id}

Obter detalhes de um documento específico

Exemplo:

GET /api.php/documents/123
Headers: X-API-Key: edoc-api-key-2024
POST
/api.php/documents

Criar um novo documento

Body (JSON):

{
  "title": "Novo Documento",
  "content": "Conteúdo do documento...",
  "category_id": 1,
  "priority": "Alta",
  "deadline": "2024-12-31"
}

Exemplo:

POST /api.php/documents
Headers: X-API-Key: edoc-api-key-2024
Content-Type: application/json

{
  "title": "Contrato de Prestação de Serviços",
  "content": "Este contrato estabelece...",
  "priority": "Alta"
}
PUT
/api.php/documents/{id}

Atualizar status de um documento

Body (JSON):

{
  "status": "Aprovado"
}

🔧 Exemplos de Integração

JavaScript (Fetch API)

const apiKey = 'edoc-api-key-2024';

fetch('http://seusite.com/api.php/documents', {
  headers: {
    'X-API-Key': apiKey,
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data));

PHP (cURL)

$apiKey = 'edoc-api-key-2024';
$url = 'http://seusite.com/api.php/documents';

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey,
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);

Python (Requests)

import requests

api_key = 'edoc-api-key-2024'
url = 'http://seusite.com/api.php/documents'

headers = {
    'X-API-Key': api_key,
    'Content-Type': 'application/json'
}

response = requests.get(url, headers=headers)
data = response.json()

⚡ Limites e Considerações

  • Rate Limiting: Máximo 1000 requisições por hora por API Key
  • Timeout: 30 segundos por requisição
  • Formato: Apenas JSON é suportado
  • CORS: Habilitado para integração com aplicações web
  • Logs: Todas as requisições são registradas para auditoria

🚨 Códigos de Erro

  • 200 - Sucesso
  • 400 - Dados inválidos
  • 401 - API Key inválida
  • 404 - Recurso não encontrado
  • 405 - Método não permitido
  • 500 - Erro interno do servidor
